Guia informativa: Bem y Camelcase

Explora las metodologias mas comunes para nombrar clases y variables en desarrollo

¿Que es BEM?

BEM es una medodologia para organizar y nombrar clases en HMTL y CSS, facilitando la creación de código más legible, reutilizable y escalable, Divide los componentes de la interfaz de usuario en tres elementos: Bloques, Elementos y Modificadores.

Ventajas de usar BEM:

  1. Mejora la legibilidad del código
  2. Facilita el mantenimiento
  3. Evita conflictos entre estilos

Ejemplos de código BEM:

  • .button
  • .button__icon
  • .button--disabled

¿Que es camelcase?

es un tipo de escritura en el que cada palabra de una palabra compuesta se escribe con mayúscula, excepto la primera. Este estilo se utiliza principalmente en programación y desarrollo web para variables, funciones y otros elementos.

¿Dónde se usa camelCase?

  1. Variables en JavaScript
  2. Funciones en lenguajes de programación
  3. Nombrar propiedades en objetos

Ejemplos de código camelCase:

  • userName
  • getUserInfo()
  • backgroundColor

Comparacion entre BEM y Camelcase

Característica BEM CamelCase
